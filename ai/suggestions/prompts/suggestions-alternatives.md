# Suggestions Alternatives Prompt

Provide 2–4 alternative approaches:

| Strategy | Description |
|----------|-------------|
| Minimal | Smallest viable change |
| Optimized | Balanced refactor + fix |
| Complete | Includes tests, types, docs |
| Aggressive | Broad architectural improvement |

For each alternative include:

- Title and one-line summary
- Pros (2–3)
- Cons (1–2)
- Estimated files and line range

Mark one alternative as recommended when appropriate.
