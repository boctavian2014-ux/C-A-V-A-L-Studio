# Roadmap 6 Luni

## Luna 1: Core + Build

- Stabilizare Electron, TypeScript, Webpack si Node 20+.
- Integrare initiala cu structura fork VS Code.
- Product configuration CAVALLO Studio.
- Workbench shell, command palette si extension host skeleton.
- CI pentru typecheck, lint si build.

## Luna 2: AI Layer

- Model router cu provider priority si fallback.
- Chat panel si composer.
- Agenti refactor, debug si reasoning.
- Streaming responses si credential storage.
- Primele evaluari pentru modele frontier si fallback open-source.

## Luna 3: Context Engine

- Indexare incrementala.
- Embeddings reale si cache local.
- Vector DB persistent.
- Semantic search in UI.
- Dependency graph vizual si API pentru agenti.

## Luna 4: Marketplace

- Marketplace server MVP.
- Client marketplace in IDE.
- Instalare/dezinstalare extensii.
- Compatibilitate extensii VS Code.
- Semnare, verificare si first-party extensions.

## Luna 5: Romania Layer

- Localizare UI RO completa.
- ANAF CUI/TVA lookup.
- eFactura validate/submit workflows.
- ONRC search.
- Education Mode cu explicatii in romana.

## Luna 6: Polish + Beta

- Performanta workbench si startup.
- QA pe Windows, macOS si Linux.
- Telemetry privacy controls.
- Theme polish si onboarding.
- Beta privata pentru dezvoltatori romani si echipe pilot.
