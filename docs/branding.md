# Branding

Branding-ul Caval Studio este definit in `branding/`:

- `brandbook.md` descrie identitatea, tonul, culorile si principiile UI.
- `logo-directions.md` propune directii de logo.
- `theme-dark.json` defineste tema dark premium.
- `theme-light.json` defineste tema light.

## Directie

Caval Studio trebuie sa para premium si local in acelasi timp: un produs tehnic matur, cu un strat romanesc discret si util. Editorul ramane concentrat, iar AI-ul apare ca infrastructura de lucru, nu ca decor.
