# Build Instructions

## Cerinte

- Node.js 20 sau mai nou.
- npm 10 sau mai nou.
- Windows 10/11, macOS sau Linux.

## Instalare

```bash
npm install
```

## Verificare TypeScript

```bash
npm run typecheck
```

## Build productie

```bash
npm run build
```

## Pornire Electron

```bash
npm start
```

## Dezvoltare

```bash
npm run dev
```

Webpack produce bundle-uri pentru Electron main, renderer si servicii Node in `dist/`.
