# Installer

Documentatia completa pentru installer este in `installer/docs/installer.md`.

## Rezumat

- Windows: NSIS `.exe` si `.msi`.
- macOS: `.dmg` si `.pkg`, code signing si notarizare.
- Linux: `.deb`, `.rpm`, `.AppImage`.
- Auto-update: stable, beta, nightly.
- Crash reporting: Sentry/custom endpoint.
