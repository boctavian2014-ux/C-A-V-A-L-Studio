# Romania Layer

Romania Layer adauga functionalitati locale direct in IDE:

- `localization/ro.json` pentru UI in limba romana.
- `anaf-api.ts` pentru verificari CUI/TVA.
- `efactura.ts` pentru validare si trimitere eFactura.
- `onrc.ts` pentru cautari companii.
- `education-mode.ts` pentru explicatii si hint-uri in romana.

## Principii

- Fara credentiale hardcodate.
- Fluxurile fiscale trebuie logate auditabil.
- UI-ul romanesc trebuie sa fie natural, nu traducere mecanica.
- Education Mode trebuie sa invete utilizatorul, nu sa ascunda complet problema.
