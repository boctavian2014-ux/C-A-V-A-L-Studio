# Logo Directions

Conceptul central: verticalitate, fluiditate, tehnologie si premium. Logo-ul foloseste o silueta verticala stilizata de caval, 1-3 linii paralele si o forma geometrica subtila: cerc, patrat rotunjit sau capsula verticala.

## Directia A: Monoline Vertical

- 2-3 linii verticale.
- O linie centrala mai lunga, doua linii laterale mai scurte.
- Forma geometrica subtila in jur.
- Cea mai potrivita pentru primary logo si sidebar.

## Directia B: Caval Abstract

- Silueta abstracta de caval.
- Forma organica, dar geometrica.
- Vibe artistic + tech.
- Potrivita pentru splash screen si materiale de brand.

## Directia C: Pulse Tech

- Linie verticala cu gradient cyan.
- Efect de puls AI.
- Potrivita pentru app icon si AI panel.

## Variante

- **Primary Logo:** simbol + text `CAVALLO Studio`.
- **Secondary Logo:** doar simbol.
- **App Icon:** simbol intr-un patrat rotunjit.
- **AI Icon:** simbol cu glow Cyan Pulse.

## Logo Grid

- Grid: 8px.
- Stroke: 2-3px.
- Spacing intre linii: 4-6px.
- Raport simbol:text: 1:2.
- Inaltime simbol = inaltimea literei `C` din `Caval`.

## Aplicare UI

- App Icon: simbol + glow cyan.
- Splash Screen: simbol mare centrat pe Deep Blue.
- Sidebar: simbol mic, monoline, Soft White.
- AI Panel: simbol cu puls cyan.
- Marketplace: simbol Carpathian Gold.
