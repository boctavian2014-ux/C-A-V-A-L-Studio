# CAVALLO Studio Brandbook

CAVALLO Studio este un IDE romanesc premium pentru dezvoltatori care vor viteza, claritate si AI frontier fara sa piarda controlul asupra codului. Identitatea vizuala este dark tech, cu Cyan Pulse, linii verticale inspirate din caval si radacini romanesti discrete.

## Identitate

- **Personalitate:** calm, precis, premium, tehnic, natural.
- **Promisiune:** un flow de lucru fluent intre editor, AI, context si instrumente locale pentru Romania.
- **Ton:** romanesc modern, fara rigiditate corporate, cu respect pentru meserie.
- **Public:** senior engineers, startup teams, agent builders, studenti avansati si companii care lucreaza cu fiscalitate romaneasca.
- **Mesaje:** "Flow natural in cod.", "Respira. Scrie. Creeaza.", "AI care simte ritmul proiectului.", "Puterea codului, simplitatea cavalului."

## Directie Vizuala

- **Dark first:** Graphite Black si Deep Blue sunt fundalurile implicite.
- **Premium tech:** monoline, geometrie curata, spacing generos, glow cyan subtil.
- **Romania Layer:** radacini romanesti discrete, fara simbolistica folclorica explicita.
- **Flow natural:** tranzitii line, panouri aerisite, animatii utile.
- **Verticalitate:** simbolul Caval foloseste 1-3 linii verticale ca ritm vizual.

## Culori

- Graphite Black: `#0E0E0F`
- Deep Blue: `#0A1A2F`
- Cyan Pulse: `#00E0FF`
- Soft White: `#F5F7FA`
- Carpathian Gold: `#D4A857`
- Forest Green: `#2FBF71`
- Mountain Grey: `#8A95A6`

## Logo

- Simbolul se construieste pe grid de 8px.
- Grosime linii: 2-3px.
- Spacing intre linii: 4-6px.
- Raport simbol:text: 1:2.
- Inaltimea simbolului trebuie sa egaleze inaltimea literei `C` din `Caval`.
- Variante: Primary Logo, Secondary Logo, App Icon, AI Icon.

## Principii UI

- Editorul ramane centrul produsului.
- AI-ul este un partener contextual, nu un overlay agresiv.
- Marketplace-ul evidentiaza extensiile verificate si compatibilitatea VS Code.
- Romania Layer este utilitar, integrat in comenzi si panouri, nu intruziv.
- Cyan glow se foloseste pentru focus, AI state si momente de orientare, nu ca decor permanent.
